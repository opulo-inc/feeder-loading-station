# Feeder Loading Station

The Feeder Loading Station is an accessory used to help load electronic component tape into LumenPnP Feeders.

photo

The FLS is effectively a short length of extrusion with a powered slot, used to provide a dedicated station to power the feeder while loading in component tape. It accepts a standard 24v barrel jack (same as what's used with the LumenPnP) and provides power to the single slot on the rail.
